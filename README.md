## Features
- Adding expenses
- Deleting expenses
- Viewing expenses
- Automatic local saving

## Installation
```bash
$ pip install textual
$ git clone https://github.com/Limit-sest/Expense_tracker.git
```

## Usage
### Running the app
```bash
$ cd Expense_tracker
$ python3 main.py
```

### Using the app
- Press `a` to add an expense while in the list view
- Press `d` to delete a selected expense while in the list view
